# smart_wardrode
PYTHON = 3.11.14

test_images папка содержащая тестовые изображения которые нужно обрабоать.
При запуске скрипта run_model.py модель проходит по указанной папке и сохраняет результат в results.json

В test_model.py изображения классифицируются, размечаются bbox и сохраняются.

В папке preparing_dataset находятся скрипты которые нужны для подготовки датасета в Yolo-формат и его проверки.
